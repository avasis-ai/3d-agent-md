# 3D-Agent.md (#54)

## Tagline
Autonomous agents that model, rig, and animate directly in Blender.

## What It Does
Connecting an LLM directly to Blender's Python API via a specialized SKILL.md, users prompt create a low-poly medieval tavern. The agent sequentially executes the complex Python commands to generate the geometry and materials natively in the software.

## Inspired By
Blender Python API, MCP, Multimodal + 3D generation

## Viral Potential
Extremely visual, highly shareable outputs bridge AI and digital art. 3D modeling has a massive learning curve; this democratizes creation. Bridges AI with the massive Blender open-source community.

## Unique Defensible Moat
An exhaustive, highly tuned library of procedural generation scripts mapped to semantic intents bypasses the LLM's inability to understand complex 3D spatial geometry natively.

/blender-plugin, /agent-core, GPLv3, sample generation script
