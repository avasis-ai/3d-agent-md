<h3>Autonomous agents that model, rig, and animate directly in Blender.</h3>

<i>Connecting an LLM directly to Blender's Python API via a specialized SKILL.md, users prompt create a low-poly medieval tavern. The agent sequentially executes the complex Python commands to generate the geometry and materials natively in the software.</i>
